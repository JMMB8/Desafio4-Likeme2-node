## Likeme

## Instrucciones:
### Database PostgreSQL 17
1.- ```CREATE DATABASE likeme;```<br>
2.- ```CREATE TABLE posts (id SERIAL, titulo VARCHAR(25), img VARCHAR(1000),
descripcion VARCHAR(255), likes INT);```<br>

## Backend
1.-```cd backend```<br>
2.-```npm install```<br>
3.-```npm run dev```<br>
